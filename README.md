# Freight Marketplace MVP

International specialty freight marketplace connecting shippers and shipping companies.

## Features

**Shippers:**
- $1 to list a shipment (covers the first month live)
- $5/month to keep a posting active after the first month
- $10/month to browse the carrier directory
- Direct contact with companies via inquiry form

**Shipping Companies:**
- $50/month subscription for unlimited access
- Browse all active shipments
- Contact shippers about shipments they're interested in

## Tech Stack

- **Backend:** Python stdlib (http.server + sqlite3)
- **Frontend:** Vanilla JavaScript + HTML + CSS
- **Database:** SQLite
- **Payments:** Stripe (integration in progress)
